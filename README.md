# GoStack Nivel 02 Primeiro Projeto Node

![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat-square)

> Nessa etapa vamos continuar os aprendizados com Node.js aproveitando o projeto desenvolvido até agora para começar a trabalhar com conceitos de banco de dados, autenticação, autorização, etc.

## Tecnologias

- TypeScript

## Como rodar

### Pré-requisitos

- Node.js 18+ e npm

### Instalação

```bash
git clone https://github.com/salomaocalheiros/GoStack--Nivel-02---primeiro-projeto-node.git
cd GoStack--Nivel-02---primeiro-projeto-node
npm install
```

### Execução

```bash
npm start
```
